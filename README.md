# React + Node Project

A simple app with a React front end loaded from the browser and a Node.js/Express backend.

## Run locally

1. Open a terminal in `react-node-app`
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`

## Files

- `server.js` - Express server that serves the React page
- `public/index.html` - React app entry page
- `public/app.js` - React component and API fetch example
